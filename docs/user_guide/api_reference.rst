Article
-------

.. _article shortcut:

Article objects can also be created with the shortcut method::

    a = newspaper.article(url, language='en', ...)

which is equivalent to::

    a = newspaper.Article(url, language='en', ...)
    a.download()
    a.parse()

You can pass any of the Article constructor arguments to the shortcut method.
